## 2. 사용자 결정 분석 (왜 $4.99/$49.99인가)

### 2.1 옵션 비교 (AskUserQuestion 4 선택지)

| 옵션 | 가격 | positioning | 사용자 선택 |
|---|---|---|---|
| A | $6.99/mo · $59.99/yr | Memrise Pro 수준, premium 가성비 | ❌ |
| B | $7.99/mo · $69.99/yr | 중간, premium positioning (recommended) | ❌ |
| C | $9.99/mo · $79.99/yr | LingoDeer 직접 경쟁 | ❌ |
| **D** | **$4.99/mo · $49.99/yr** | qa placeholder, premium 진입가 | **✅ 사용자 선택** |

### 2.2 사용자 선택 합리성 분석

사용자는 권고된 옵션 B(중간)가 아닌 옵션 D(가장 낮은) 선택. orchestrator 해석:

1. **qa placeholder와 정합**: M4 entry preview dispatch v0(W17-Q1)에 이미 $4.99 기록. 추가 SSOT 변경 0 → **정합성 비용 최소**
2. **$1.99 대비 2.5배 인상**: 사업계획서 v0.1 거부했지만 인상 폭이 충분 → "싸구려 인식 회피" 충족
3. **경쟁사 평균 아래**: Memrise $8.49, LingoDeer $14.99 대비 여전히 낮음 → 진입 장벽 최소화 유지
4. **1인 운영 고려**: 베타 1~2명 결제로도 break-even 6~12명 도달 가능 (사업계획서 §14.2 재계산)

### 2.3 가격 인상 여지 보존

D-018은 "GA 초기 가격"으로 봉인. 사업계획서 §10.2 재조정 정책:
- 콘텐츠 확장 (starter 60 → core 180 → topik) 후 가격 인상 검토
- 유료 전환율 낮으면 무료 한도 조정 우선 (가격 인하 X)
- 연 구독 비중 낮으면 연간 가격 조정 ($49.99 → $44.99)

따라서 $4.99는 "출시가"이며 1~2년 내 $6.99~$8.99로 인상 trajectory 가능.

## 3. 손익 재계산 (D-018 봉인 후)

### 3.1 Break-even

| 항목 | 사업계획서 v0.1 ($1.99) | D-018 ($4.99) | 변화 |
|---|---|---|---|
| 사용자 결제 후 net | $1.39/mo (Apple 30% 제외) | **$3.49/mo** | +$2.10/user/mo |
| 월 운영 burn | $20~40 | $20~40 | 동일 |
| Break-even 유료 사용자 | 14~28명 | **6~12명** | **절반** |

### 3.2 베타 30명 시나리오

- 베타 사용자 30명 × paywall_view_to_purchase 4% target = **1~2명 유료 전환**
- $4.99 × 1.5명 평균 × net 70% = $5.2/mo
- 베타 기간 동안 break-even 미달이지만 GA 후 1k MAU 도달 시 break-even 가능

### 3.3 GA 후 1k MAU 시나리오

- 1,000 MAU × 4% paywall conversion = **40 유료 사용자/mo (목표)**
- 40 × $3.49 = **$139/mo net** → 운영 burn 충분히 cover + 콘텐츠 batch 2 (core 180) 투자 여력

## 4. paywall_view_to_purchase threshold 영향 평가

PRD §8.2 target 4% / minimum 2% — D-018 봉인 후 영향:

### 4.1 Price sensitivity 측면

- $1.99 → $4.99 2.5배 인상
- 일반 가격 탄력성 0.5~1.0 가정 시 conversion rate 1.25~2.5x 하락 가능
- baseline에서 4% → 1.6~3.2% 범위 가능

### 4.2 Premium positioning 측면

- 낮은 가격은 "장난감" 신호 → 진지한 학습자 회피
- $4.99는 "진지한 학습 도구" 신호 → 진지한 학습자 유입 증가
- 자기 선택 효과로 paywall_view 도달 자체가 더 높은 결제 의향 cohort

### 4.3 종합 권고

target 4% / minimum 2% **유지**. W16 14d baseline 수집 후 relative (mean−1.5σ) 전환 시 재조정.

미달 시 (D-018 §재조정 정책):
- Conditional Pass (≥2%): 무료 한도 조정 우선 (예: starter 60 → 90으로 무료 확대)
- Fail (<2%): 가격 인하 검토 또는 베타 한정 할인 (R-M5-01 §3.9 응답 반영)
