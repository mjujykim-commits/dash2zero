## 2. 핵심 산출물 상세

### 2.1 화면 (frontend + designer)

- **Onboarding**: 4 카테고리 라디오 (K-pop / K-drama / Travel / Other). 가입 강제 안 함 (게스트 모드).
- **Home**: 단일 화면 단일 CTA (DESIGN_DIRECTION §8.1). "Resets at 4:00 AM" 캡션 (UX-DOC-012). Streak/Mastered understated 처리 (DESIGN_DIRECTION §6 — Quiet 톤).

### 2.2 Audio (frontend)

`src/lib/audio.ts`:
- `initAudio()` — AVAudioSession `playsInSilentModeIOS=true` (Q-FE-NEW-003 결정)
- `playAudio(uri)` — 단일 sound 재생, 자동 unload, error 추적 (audio_play_failed 이벤트)
- AudioFocus ducking (다른 앱 일시정지)

### 2.3 Analytics (frontend + analytics)

`src/lib/analytics.ts`:
- `applyConsent` — Firebase setAnalyticsCollectionEnabled + Crashlytics 동시
- ESSENTIAL_EVENTS 화이트리스트 (동의 전 송신 가능: app_open / age_gate_completed / privacy_choices_completed)
- TraceCollector 인터페이스 어댑터 (ADR-0002) — M3 후반에 Langfuse로 교체 가능
- IDFA 미사용 + ad personalization disabled (FE-NEW-002 결정)

### 2.4 RPCs (backend)

`0003_rpc.sql`:
- `increment_daily_usage` — atomic UPSERT, ON CONFLICT
- `compute_streak` — 사용자 로컬 04:00 calendar day 기준, 거꾸로 day-by-day 카운트
- `get_today_summary` — Home 화면 단일 호출 (new_words_remaining / reviews_due / streak_days / mastered_count / is_premium)
- 모두 `SECURITY DEFINER` + `service_role`/`authenticated` GRANT 분리

### 2.5 Edge Functions 공통 모듈 (backend, R-12 해결)

`apps/api/edge-functions/_shared/srs.ts`:
- `mobile/src/srs/leitner.ts`와 동일 로직
- Deno runtime 호환 (sibling copy)
- M3 진입 시 esm 빌드로 단일 source 통합 검토

### 2.6 Starter Pack 60단어 v1.0 (learning)

8 카테고리 분포 — 인사 10 / 음식 10 / 이동 8 / 숫자·시간 8 / 인칭 6 / 형용사 8 / 의문사 5 / 패턴 5 = 60.

각 단어:
- word_id (영구 키), korean, romanization, gloss, example_ko/en, distractors_candidate (3개)
- 검수 워크플로우 status field 포함

### 2.7 TTS 생성 스크립트 (devops + learning)

`scripts/content/generate-audio.ts`:
- yaml 입력 → 단어/예문 audio 생성 → SHA256 hash → manifest.json
- Google Cloud TTS Neural2 ko-KR-Neural2-A 호출 (실제 키는 W8 종료까지 발급)
- 출력 매니페스트로 audio_assets 테이블 import 가능
